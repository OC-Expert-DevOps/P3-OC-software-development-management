# DataShare — Secure File Transfer Platform

A secure file sharing platform for freelancers and small businesses, enabling temporary download link generation with expiration and optional password protection.

## Stack

| Layer | Technology |
|-------|-----------|
| Back-end | NestJS (TypeScript) |
| Front-end | React + Vite (TypeScript) |
| Database | PostgreSQL |
| Storage | MinIO (S3-compatible) |
| Deployment | Docker Compose |
